# Anas Semesmieh — Portfolio

Source code for **[anas.semesmieh.com](https://anas.semesmieh.com)** — my personal portfolio and engineering blog.

Built with hand-crafted HTML/CSS (no frameworks, no build pipeline) and served via Cloudflare Tunnel + Caddy reverse proxy from a self-hosted Proxmox homelab.

---

## What's in here

```
index.html              → Homepage (bio, experience, skills, contact)
about/index.html        → Full profile page
blog/index.html         → Blog index
blog/posts/*.html       → Individual blog posts (25 posts)
sitemap.xml             → Sitemap (30 URLs)
robots.txt              → Search engine directives
anas-semesmieh-resume.pdf → Current resume (linked from the site)
```
